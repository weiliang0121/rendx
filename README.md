# dye

A repository project.

## 关联本地仓库 / Link Local Repository

### 克隆仓库 / Clone Repository

使用以下命令克隆此仓库到本地：

```bash
git clone https://github.com/weiliang0121/dye.git
cd dye
```

### 使用 SSH 克隆 / Clone with SSH

如果你已配置 SSH 密钥，可以使用：

```bash
git clone git@github.com:weiliang0121/dye.git
cd dye
```

### 已有本地仓库？关联远程仓库 / Already Have a Local Repository?

如果你已经有一个本地仓库，想要关联到这个远程仓库：

```bash
# 添加远程仓库
git remote add origin https://github.com/weiliang0121/dye.git

# 或使用 SSH
git remote add origin git@github.com:weiliang0121/dye.git

# 查看远程仓库
git remote -v

# 拉取远程更新
git pull origin main

# 推送本地更改
git push -u origin main
```

### 常用 Git 命令 / Common Git Commands

```bash
# 查看状态
git status

# 添加文件
git add .

# 提交更改
git commit -m "your commit message"

# 推送到远程
git push

# 拉取远程更新
git pull

# 查看提交历史
git log
```
